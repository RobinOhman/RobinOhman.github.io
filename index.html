C++ Game Engine

A simple game engine created using C++.

It supports object creation, collision physics, sound, animation and custom key bindings.

It features simpleton patterns for audio and imagery to minimize the amount of SDL the user needs to interact with, while also preventing multiple instances of the same file being stored.
